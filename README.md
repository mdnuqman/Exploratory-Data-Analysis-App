## 📊 Exploratory Data Analysis (EDA) App

An interactive web application built with Streamlit that allows users to explore and analyze datasets without writing any code.

### Features
- Upload CSV or Excel files
- Preview data and view dataset overview
- Detect missing values per column
- Statistical summary for numerical and non-numerical columns
- Correlation heatmap
- Filter data by column value
- Data visualization (Line, Scatter, Bar, Histogram, Box Plot)
- Outlier detection using IQR method

### Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib

### How To Run
pip install streamlit pandas matplotlib openpyxl
streamlit run EDA.py
